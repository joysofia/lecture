## 課綱

++++  | +++++  | ++++++++++++++++++++++++++ |++++++++++++++++++++++++++++++++++++++++++
1    | 09/12  | Orientation   | Rstudio installation, Markdown, Datacamp custom track

3    | 09/26  | R data structures| vectors; conditionals
